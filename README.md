# 🎉 Website Chúc Mừng Sinh Nhật 🎂

Dự án nhỏ gồm các trang web HTML/CSS/JS:

- `login.html` : Trang nhập mật khẩu bằng bàn phím số.  
- `happybirthday.html` : Trang chúc mừng sinh nhật với hiệu ứng đặc biệt.  


---

## 🚀 Cách chạy

### 1. Mở trực tiếp trên máy tính
- Bấm đúp vào file `login.html` để mở trong trình duyệt (Chrome, Edge, Firefox...).  
- Nhập mật khẩu: **28092005**  
- Nếu đúng → sẽ tự động chuyển sang `happybirthday.html`.  
- Từ `happybirthday.html` có thể truy cập tiếp `anh.html`.

---

### 2. Chạy bằng **VS Code + Live Server**
1. Cài đặt extension **Live Server** trong VS Code.  
2. Mở thư mục dự án bằng VS Code.  
3. Chuột phải vào `login.html` → chọn **Open with Live Server**.  
4. Truy cập link (mặc định: `http://127.0.0.1:5500/login.html`).  
5. Nếu muốn mở trên điện thoại:  
   - Tìm địa chỉ IP của máy tính trong mạng LAN (ví dụ: `192.168.1.5`).  
   - Trên điện thoại, vào link:  
     ```
     http://192.168.1.5:5500/login.html
     ```
   - (Điều kiện: điện thoại và máy tính cùng WiFi).

---
